# Sonic-Deauther
ESP8266 Pen-testing Deauther made into a sonic screwdriver controller.
# Original Code
https://github.com/spacehuhn/esp8266_deauther

# Functions
* WiFi Deauther
* WiFi Control
* Sonic SFX, LFX
* Torch
* UV Torch
* IR Code Capture
* IR Code Emission
* Sensor Feedback
* Mode Setting
